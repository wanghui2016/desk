## Build Setup

``` bash
# install dependencies
npm install

# start app
npm start

# build
npm run build

# package for app (mac64)
npm run package:osx

# package for app (windows64)
npm run package:win


# 快捷键说明
os：mac
  command + alt + R ==》刷新页面
  command + alt + P + O ==》线上环境
  command + alt + D + V ==》dev环境
  command + alt + P + E ==》pre环境
  command + alt + L + C ==》本地端口3011环境(saas web service)
os: windows
  control + alt + R ==》刷新页面
  control + alt + P + O ==》线上环境
  control + alt + D + V ==》dev环境
  control + alt + P + E ==》pre环境
  control + alt + L + C ==》本地端口3011环境(saas web service)

  alt + d + k ==》打开页面开发
  alt + a + i ==> 弹出对话框显示当前运行环境(online，dev，pre)
